Preprocessing Needed
Lowercasing → Convert all text to lowercase for consistency.
Removing emojis → Emojis can be removed or converted to text labels.
Correcting typos → Fix spelling mistakes using dictionaries or tools.
Expanding slang/short forms → Convert “u” → “you”, “gr8” → “great”.
Removing special characters → Symbols like @, !!!, ??? cleaned.
Removing extra letters → Convert “soooo” → “so”.
Tokenization → Split sentences into individual words.
Description

This assignment highlights the challenges in processing real-world text data, which often contains slang, typos, emojis, and informal language. It focuses on identifying these issues and applying preprocessing techniques to clean and standardize text for machine learning models.

Learning Outcomes
Understand challenges in natural language processing (NLP).
Learn techniques for text preprocessing and cleaning.
Identify different types of noise in text data.
Improve ability to prepare text for ML and AI models.
Develop practical skills in handling real-world messy data.
